Полностью переработать старый скрипт fbacc.js.

Важно:
не пытаться рефакторить старую архитектуру и не пытаться “оживить” весь legacy-код. Старый скрипт содержит много устаревших, сломанных и потенциально опасных функций.

Задача:
создать современную lightweight internal utility tool в фирменном стиле FB Bookmarklet Tools, используя только полезное ядро старого fbacc.

==================================================
НОВОЕ ПОЗИЦИОНИРОВАНИЕ
==================================================

Новый инструмент должен быть:

FB Account Inspector

Это:
- utility tool;
- inspector/dashboard;
- lightweight operational helper;
- read-focused internal tool;
- не destructive multi-tool комбайн.

==================================================
ГЛАВНЫЙ ПРИНЦИП
==================================================

Скрипт должен:
- быстро показывать полезную информацию;
- помогать диагностировать аккаунты;
- помогать ориентироваться в структуре BM;
- помогать быстро копировать данные;
- помогать быстро переходить в нужные разделы.

Скрипт НЕ должен:
- выполнять опасные действия;
- использовать мутные legacy mutations;
- работать как automation exploit tool;
- содержать мёртвые функции.

==================================================
СОХРАНИТЬ И ПЕРЕРАБОТАТЬ
==================================================

Нужно сохранить и обновить:

1. Загрузка списка ad accounts
- account id
- account name
- status
- currency
- timezone
- spend limit / balance (если доступно)

2. Business Manager информация
- BM name
- BM id
- связь аккаунта с BM
- статус BM

3. Страницы
- связанные страницы
- page id
- page name

4. Быстрые действия
- copy account id
- copy BM id
- copy page id
- open account
- open BM
- open page
- open billing
- open account settings

5. Диагностика
- disabled / active
- personal / business owned
- basic health indicators
- missing permissions / errors
