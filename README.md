# FB Bookmarklet Design System

Универсальная дизайн-система для внутренних bookmarklet-скриптов Facebook Ads Manager.

## Состав

- `UI_STYLE_GUIDE.md` — главный гайд для Codex и разработчиков.
- `CODEX_PROMPT.md` — готовый промт для применения системы к новым скриптам.
- `ui/core/tokens.js` — дизайн-токены.
- `ui/core/typography.css` — типографика.
- `ui/core/styles.css` — базовые компоненты и layout.

## Смысл

Система не копирует внешний вид одного эталонного скрипта 1:1. Она извлекает общие паттерны:
- compact dark utility UI;
- fixed modal поверх Facebook;
- зелёный акцент;
- русская локализация;
- короткие actions;
- tabs при нескольких сценариях;
- обязательный лог для долгих действий;
- минимальный визуальный шум.
